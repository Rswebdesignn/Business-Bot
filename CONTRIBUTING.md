# Contributing to Business Chatbot

Thank you for your interest in contributing to the Business Chatbot project. As this is a proprietary project owned by Rohit Gunthal, contributions are by invitation only.

## Important Notice

This project is not open-source. All rights are reserved by Rohit Gunthal. Unauthorized contributions, forks, or derivatives are not permitted without explicit written permission.

## Authorized Contributors

If you have been authorized to contribute to this project, please follow these guidelines:

### Code Style

- Follow PEP 8 style guidelines for Python code
- Use 4 spaces for indentation (no tabs)
- Use descriptive variable and function names
- Add comments for complex logic
- Write docstrings for all functions, classes, and modules

### Commit Messages

- Use clear and descriptive commit messages
- Start with a verb in the present tense (e.g., "Add feature" not "Added feature")
- Reference issue numbers when applicable

### Pull Request Process

1. Ensure your code follows the style guidelines
2. Update documentation if necessary
3. Test your changes thoroughly
4. Submit a pull request with a clear description of the changes

### Development Workflow

1. Create a feature branch from `main`
2. Make your changes
3. Test your changes locally
4. Submit a pull request to `main`
